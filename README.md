#  WanderWay - Travel Website 
#  Live-Link: https://wanderway-travel-website.onrender.com

WanderWay is a modern and responsive travel website designed to showcase popular destinations, travel packages, and provide users with an engaging browsing experience. Built with **HTML, CSS, and JavaScript**, it features interactive cards, smooth scrolling, and a clean UI.

---

## Features

- **Responsive Design:** Works seamlessly on desktop, tablet, and mobile devices.
- **Popular Destinations:** Highlighting top destinations with images, descriptions, ratings, and price.
- **Interactive Cards:** Smooth auto-scroll for destination cards.
- **Navigation Menu:** Toggleable mobile menu for better accessibility.
- **User-Friendly Interface:** Clean layout with clear calls-to-action.
- **Modern Styling:** Attractive color scheme, fonts, and hover effects.

---

## Technologies Used

- **HTML5** – Structure of the website
- **CSS3** – Styling, Flexbox, Grid, responsive design
- **JavaScript** – Interactive cards, auto-scroll functionality
- **Responsive Design** – Mobile-first approach
